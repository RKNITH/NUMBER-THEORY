Number Theoretic Functions

Introduction to Number Theoretic Functions

Number theoretic functions are functions defined on the set of natural numbers and have properties that are significant in number theory. These functions include divisor functions, Euler's totient function, and many others.

Divisor Functions

Divisor functions are functions that count the number of divisors or sum the divisors of a given integer.

σk(n) - Sum of Divisors Function

The sum of divisors function, denoted as σk(n), is defined as the sum of the k-th powers of the positive divisors of n. For k = 0, σ0(n) represents the number of divisors of n, also denoted as d(n).

Properties

  • σk(1) = 1 for all k.
  • σk(n) is a multiplicative function.
  • If p is a prime and e is a positive integer, then σk(pe) = 1 + pk + p2k + ... + pek.

Sum of k-th Powers of All Divisors of a Number

The sum of the k-th powers of all divisors of a number n, denoted as σk(n), is a useful function in number theory.

Definition

For a positive integer n, σk(n) is defined as:

σk(n) = Σ dk

where the sum is over all positive divisors d of n.

Properties

  • σ0(n) = d(n), which is the number of divisors of n.
  • σ1(n) = σ(n), which is the sum of all divisors of n.
  • For any prime p and positive integer e, σk(pe) = 1 + pk + p2k + ... + pek.
  • σk(n) is a multiplicative function, meaning if m and n are coprime, then σk(mn) = σk(m)σk(n).

Euler's Totient Function

Euler's totient function, denoted as φ(n), counts the number of integers up to n that are relatively prime to n.

Definition

For a positive integer n, φ(n) is defined as the number of integers k in the range 1 ≤ k ≤ n for which gcd(n, k) = 1.

Properties

  • φ(p) = p - 1 if p is a prime.
  • φ(mn) = φ(m)φ(n) if gcd(m, n) = 1.
  • φ(pk) = pk - pk-1 for a prime p and a positive integer k.

Mobius Function

The Möbius function, denoted as μ(n), is an important function in number theory and combinatorics.

Definition

  • μ(n) = 1 if n is a square-free positive integer with an even number of prime factors.
  • μ(n) = -1 if n is a square-free positive integer with an odd number of prime factors.
  • μ(n) = 0 if n has a squared prime factor.

Properties

  • μ(1) = 1.
  • The Möbius function is a multiplicative function.
  • The sum of the Möbius function over the divisors of n equals 0 for n > 1.

Liouville Function

The Liouville function, denoted as λ(n), is defined for all positive integers n and gives (-1) raised to the power of the number of prime factors of n, counted with multiplicity.

Definition

For a positive integer n with prime factorization n = p1e1p2e2...pkek, the Liouville function is defined as:

λ(n) = (-1)e1 + e2 + ... + ek.

Properties

  • λ(1) = 1.
  • λ(p) = -1 for any prime p.
  • λ is a completely multiplicative function.
